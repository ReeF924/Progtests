# Progtest & Semestral Projects (FIT CTU)

This repository contains my solutions to programming assignments and semestral projects developed during my studies at the **Faculty of Information Technology, Czech Technical University in Prague (FIT ČVUT)**.

Most assignments in this repository were evaluated by **Progtest**, an automated system used at FIT.
For this reason every assignment solution must be within one file.

## Course Overview

### Semester 4 (Current)
* **BI-OSY (Operating Systems):** System programming in **C++**. Focus on multi-threading, synchronization (mutexes, semaphores, condition variables), and process management in UNIX.
* **BI-AG2 (Algorithms and Graphs 2):** Advanced graph algorithms.
* **BI-DAS (Data Structures):** Advanced structures like (a,b)-Trees, hashing strategies, self-balancing trees, bloom filters, heaps etc..

### Semester 3
* **BI-AG1 (Algorithms and Graphs 1):** Fundamental graph theory. Implementation of BFS, DFS, Dijkstra, MST (Kruskal/Jarník), and greedy algorithms with focus on complexity analysis.

### Semester 2
* **BI-PA2 (Programming and Algoritmization 2):** Advanced **C++ (OOP)**. Deep dive into polymorphism, templates, STL containers, and RAII.

### Semester 1
* **BI-PA1 (Programming and Algoritmization 1):** Foundations of the **C language**, low-level memory management, and recursion.
